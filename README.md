# Solum Practical Test — Solutions

This repository contains my solutions for the **Solum Practical Questions** — divided into three parts: **A, B, and C**.

---

## A. Mystic Waves

**Language:** C++
Simulates alternating magical energy waves and computes the total energy after `n` waves.

* Implements simple arithmetic logic for alternating signs.
* Handles multiple test cases efficiently.

Run:

```bash
make run
```

---

## B. CargoCraft Fleet

**Language:** C++
Determines the **minimum** and **maximum** number of crafts that can form a total of `n` propulsion units, given:

* Type A: 4 units
* Type B: 6 units

If no valid combination exists, outputs `-1`.

Run:

```bash
make run
```

---

## C. Frontend Test — Login Page

**Framework:** React + TypeScript
A responsive **login interface** with full client-side validation.

* Email & password checks
* Eye icon to show/hide password
* Disabled copy/paste for password field

Run locally:

```bash
npm install
npm run dev
```
